# Matvey Matveev — Portfolio

Personal portfolio website for a Technical Game Designer / Gameplay Designer.

## Stack

- HTML, CSS, JavaScript
- Bilingual UI: English (default) and Russian

## Local preview

```bash
python -m http.server 8765
```

Open http://localhost:8765/

## Deploy

Upload `index.html`, `styles.css`, and `script.js` to any static hosting (GitHub Pages, Beget, REG.RU, etc.).

For GitHub Pages: enable Pages in repository settings and use the `main` branch root.
